# liri-node-app
_Language_ interpretation and recognition Interface

This node application use three API's;
    - node-spotify-api
    - OMDB api
    - Bandsintow API

Search can be made by typing the four preset commands:
    -concert-this
    -spotify-this-song
    -movie-this
    -do-what-it-says

along with the second parameter on command line.
